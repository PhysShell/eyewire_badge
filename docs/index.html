EyeWire Stats Widget

A drop-in <eyewire-stats> Web Component for public EyeWire player stats. Dark sci-fi HUD card, Shadow-DOM isolated, no login required.

card — full card, one per period

mini & badge — compact inline variants

states — graceful loading / error handling

Use it

<script src="https://USERNAME.github.io/REPO/eyewire-widget.js" defer></script>

<eyewire-stats user="SomeEyeWireUser" mode="card" period="forever"></eyewire-stats>

Stats come straight from the public EyeWire API (eyewire.org/1.0/player/{user}/stats), which sends Access-Control-Allow-Origin: * — so no proxy is needed. See the README for attributes, the optional Cloudflare Worker proxy, and security notes.
